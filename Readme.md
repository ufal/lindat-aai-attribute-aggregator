# Attribute aggregator

## Requirements

- node.js + process manager e.g., pm2
- redis

## Startup

- start redis
- start


## Installation

1. Install Node/npm

2. Clone this repository
    
```
npm install --no-bin-links
```

4. Start
    
```
npm start
```



## Install new modules and save them to package.json

```
npm install --save XXX
```

## Update modules

```
npm install -g npm-check-updates
npm-check-updates -u
npm install 
```
